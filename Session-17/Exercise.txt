1. Πως μπορούμε να κάνουμε ένα Request σε ένα εξωτερικό API χωρίς να γνωρίζει ολόκληρο το url μας;

-> Οχι, δεν μπορούμε να κάνουμε ένα Request σε ένα εξωτερικό API χωρίς να γνωρίζουμε ολόκληρο το url. Για να κάνουμε ένα Request για ένα resource στο Web απαιτεί την πλήρη διεύθυνση URL(Uniform Resource Locator) αυτού του resource. Μια διεύθυνση URL περιέχει όλες τις απαραίτητες πληροφορίες για ανάκτηση και τον εντοπισμό του resource, συμπεριλαμβανομένου του πρωτοκόλλου (π.χ. HTTP), την διεύθυνση IP ή το όνομα του domain καθώς και το Path του resource στον σερβερ.
Χωρίς να γνωρίζουμε ολόκληρο το url δεν είναι μπορούμε να κάνουμε Request του αιτήματος σε ένα εξωτερικό API.

2. Αν κάνω ενα GET request και ζητήσω ενα jpeg image, το response τι content-type πρεπει να εχει;

3. Πως μπορώ να κάνω ένα GET request και το περιεχόμενο του να κρατηθεί στην cache για 1 μέρα;

4. Μπορώ να έχω μια εφαρμογή που κάνει μόνο GET requests;

5. Μπορώ να έχω μια εφαρμογή που κάνει μόνο POST requests;

6. Τι status code περιμένω από ένα API αν δεν έχω στείλει ένα
υποχρεωτικό πεδίο;

7. Τι status code περιμένω από ένα API αν δεν υπάρχει επικοινωνία με τη βάση;

8. Μπορώ να κάνω Login με GET request;

9. Ποια είναι η διαφορά ανάμεσα στο PUT και στο POST;

10. Ποια μέθοδο θα πρέπει να χρησιμοποιήσω για να «ανεβάσω» ένα αρχείο;
